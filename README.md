# 🪨📄✂️ Rock Paper Scissors (with Sound!)

A classic Rock Paper Scissors game built in Python for the Command Prompt. This version includes **audio feedback** (beeps) for every move!
Note: It only works on WINDOWS because i used winsound

## 🚀 Features
- **Interactive Gameplay:** Play against the computer in a loop.
- **Audio Feedback:** Uses the `winsound` module to beep on every choice.
- **Input Validation:** Handles typos so the game doesn't crash if you mistype "rock".
- **Lightweight:** Single-file script, no heavy libraries to install.

## 🛠️ How to Run
1. **Download** the `rock_paper_scissors.py` file.
2. **Open CMD** (Command Prompt) on Windows.
3. **Navigate** to your download folder.
4. **Run the game**:
   ```bash
   python rock_paper_scissors.py
